# Local Copy AI

An AI Website Copy Generator designed specifically for local businesses. This tool enables business owners to generate high-converting, authentic UGC (User-Generated Content) website copy and social media ad packs tailored to their specific niche.

## Features

- **Prompt Generator Tool:** Automatically construct highly engineered master prompts combining your local business details into a rigorous set of constraints. You can directly copy and paste this into ChatGPT or Gemini for perfect results.
- **Simulated Preview Engine:** View a mock output displaying the tone, structure, and aesthetic of the final requested copy before running the raw prompt.
- **UGC Pack Included:** Instructs the AI to output multiple video hooks, a 30-second ad script, and platform-specific organic captions (Instagram, TikTok, Facebook).
- **Beautiful Interface:** Built with a custom "watercolor storybook" aesthetic transitioning to a sleek dark-mode results card for maximum readability.

## How to Use

Since this application is built completely utilizing client-side technologies, no complex setup or backend server is required!

1. Clone, download, or deploy the repository.
2. Open `index.html` in your favorite web browser. 
3. Fill out your business details (Name, Niche, Value Proposition, Target Audience).
4. Click **Generate Copy Output**.
5. Switch between the Simulated Output and the Master Prompt System tabs.
6. Click the **Copy Text** button to transfer the optimized prompt directly to your clipboard.

## Technology Stack

- **HTML5**
- **Vanilla CSS**
- **Vanilla JavaScript** 
- **Marked.js** (loaded via CDN) to dynamically parse the Markdown structures into clear HTML.
